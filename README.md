# Vanguard Modding

Vanguard Modding publishes game-specific modding harnesses and blueprint docs for games that do not ship with a real modding surface.

Website: <https://vanguardmodding.com>

This repository is the source home for Vanguard's public harness work:

- game-specific runtime harnesses
- player-facing setup and troubleshooting docs
- mod-author SDKs, templates, and scaffold scripts
- reverse-engineering and bootstrap tooling where needed

Current games:

- [`games/the-farmer-was-replaced`](./games/the-farmer-was-replaced): the first published Vanguard Modding harness

Vanguard Modding is not trying to replace Nexus Mods, Steam Workshop, CurseForge, Thunderstore, mod.io, or other distribution platforms. The purpose of this repo is to make unsupported games moddable faster and give both players and authors a usable starting point.
